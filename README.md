# Laptop-prices
This Power BI project presents an interactive analysis of laptop prices using various specifications like brand, processor type, RAM, storage capacity, and operating system.  The dashboard allows users to compare prices, identify cost-driving factors, and explore pricing trends across different laptop categories.

# Project Overview
This project presents an interactive **Power BI dashboard** that analyzes laptop prices based on key specifications such as brand, processor, RAM, storage, operating system, and graphics.  
The goal is to understand how different features impact laptop pricing and to identify trends across various laptop categories.
# Key Insights
- Price comparison across different laptop brands
- Impact of RAM, storage, and processor on pricing
- Identification of high-end and budget laptop segments
- Distribution of prices based on operating system and graphics type
# Tools Used
- **Power BI**
- **Excel / CSV Dataset**
- Data Cleaning & Visualization
# Dataset Information
The dataset includes:
- Brand
- Processor type
- RAM size
- Storage type & capacity
- Operating System
- Graphics
- Price
